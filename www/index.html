Description

Webserv is a software that processes requests from clients (typically web browsers) and serves web content, such as HTML pages, images, videos, and other resources. It communicates using the HTTP (Hypertext Transfer Protocol).

How it works?

Client Request - A client (typically a web browser) sends an HTTP request to Webserv 1.O

Processing - Webserv 1.O interprets the request, locates the requested resource, or forwards the request to an application (CGI, PHP, Python scripts).

Response - Webserv 1.O returns the requested resource or an error message (ex 404 Not Found) to the client.
